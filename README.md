# 健身达人 (FitnessPro) - 智能健康管理系统

一个基于 Flask 的全栈智能健身与饮食管理平台，旨在帮助用户科学管理身材、追踪健康数据，并提供 AI 驱动的饮食识别与专业建议。

## ✨ 核心功能

*   **👤 个人资料管理**
    *   实时计算 BMR (基础代谢率) 和 BMI (身体质量指数)。
    *   支持头像上传与个人身体指标（身高、体重、年龄、性别）持久化。
    *   全站数据同步，确保各项分析基于最新的身体素质。
*   **🍱 智能饮食追踪**
    *   **AI 食物识别**：整合百度 AI 接口，支持图片识别食物并自动计算热量。
    *   **每日摄入统计**：清晰展示早、中、晚、加餐的热量分布。
    *   **热量盈余分析**：对比基础代谢与运动消耗，直观展示每日能量差。
*   **📈 体重可视化管理**
    *   **动态曲线图**：使用 ECharts 实现美观的体重变化趋势视图。
    *   **高清导出**：支持一键下载高清体重曲线图片，适配移动端显示。
    *   **智能重置**：支持一键清空历史曲线，开启新阶段训练。
*   **📅 打卡与日历**
    *   **可视化日历**：按月展示打卡记录。
    *   **详情穿透**：点击日历具体日期，可弹出暗色毛玻璃面板查看当天的饮食、运动及热量汇总。
*   **🤖 AI 健康助手**
    *   **专业建议**：基于近 7 天的数据，由智谱 AI 提供个性化的饮食与健身建议。
    *   **实时咨询**：内置基于大模型的健康助手聊天窗口，随时解答健康疑问。

## 🛠️ 技术栈

*   **后端**: Python + Flask
*   **数据库**: MySQL + SQLAlchemy (ORM)
*   **前端**: HTML5 + Vanilla CSS (Glassmorphism 风格) + Tailwind CSS (部分组件)
*   **可视化**: Apache ECharts
*   **AI 服务**: 智谱 AI (ChatGLM系列), 百度 AI (菜品识别)
*   **邮件服务**: Flask-Mail (用于发送验证码)

## 🚀 快速启动

### 1. 环境准备

建议使用 Python 3.9+ 环境。

```bash
# 创建并激活环境 (推荐使用 conda)
conda create -n fitness-env python=3.10
conda activate fitness-env

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置说明

修改项目根目录下的 `config.py` 文件：

*   **DATABASE**: 配置您的 MySQL 连接字符串。
*   **MAIL**: 配置用于发送验证码的 SMTP 账号。
*   **AI API**: 填入您的 `ZHIPU_API_KEY`、`BAIDU_API_KEY` 和 `BAIDU_SECRET_KEY`。
